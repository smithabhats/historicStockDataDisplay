# historicStockDataDisplay
This is a Java Spring Boot application that is used to display/update historic stock data

To setup:
Download the files in this repository
Setup a SpringBoot web application in Eclipse IDE
Install Postgresql 9
Run the queries in DB_queries.sql file against the database to create the tables and populate them
Clean and build the project in eclipse and run it on your localhost. Context path is http://localhost:8080/RBC/stockIndex/


The application currently takes index stocks for a limited period. This can be changed as needed
Graphs to be included for individual stock, different market sectors, and for monthly and yearly records
More analytics can be supported by the application
